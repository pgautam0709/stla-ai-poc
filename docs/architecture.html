Key design principle: All DTC and Compare table data is pre-processed at build time into static JS modules. The deployed app has zero server-side dependencies — it is a pure static SPA served from Azure CDN. The REST ingestion layer feeds a CI/CD pipeline; the pipeline output is a rebuilt, redeployed bundle.

Abstraction boundary: Legacy OEM systems never call the frontend directly. All data enters through the REST ingestion API, which normalises, validates, and stores files before triggering the processing pipeline.
